# antique — Ready-to-use generation prompts

> Copy-paste prompts for an image model (Midjourney / DALL·E / SDXL / Flux).
> Keep the gradient exact: 135° linear from #4d8dfd (blue) to #b98cff (violet).
> After generating raster, vectorize the icon (see AGENT-INSTRUCTIONS.md).

## LOGO — Concept A (primary, app icon)

```
A modern minimalist app icon for a developer tool named "antique", a
self-hosted anti-detect browser. Rounded-square (squircle) tile with a smooth
135-degree diagonal gradient from cool blue #4d8dfd to soft violet #b98cff.
Centered white glyph: a lowercase letter "a" whose inner counter is shaped like
a domino privacy eye-mask, drawn as one continuous confident monoline. Flat
vector, crisp edges, generous padding, subtle soft ambient shadow beneath the
tile, dark charcoal background #0b0e14. Premium developer-tool aesthetic like
Linear, Vercel, Raycast. No text other than the integrated "a". Legible at very
small sizes. --style flat --no photorealism, skeuomorphism, drop shadow on the glyph
```

## LOGO — Concept B (layered identities, alt)

```
Minimalist monoline logo: three overlapping rounded profile cards fanned out
like a stack of identities, each a slightly different tint along a blue-to-violet
gradient (#4d8dfd to #b98cff). Implies many isolated browser identities. Flat
vector, white/gradient on dark #0b0e14, lots of negative space, geometric,
balanced. No text, no faces.
```

## LOGO — Concept C (fingerprint ring, alt)

```
Minimal geometric fingerprint swirl inside a thin circle, monoline, where two

Blue-to-violet gradient stroke (#4d8dfd to #b98cff) on dark #0b0e14. Clean,
restrained, tech-brand. No text.
```
